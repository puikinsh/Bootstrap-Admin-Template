Free Admin Template with Twitter Bootstrap
------------------------------------------

### Installation

    $ git clone https://github.com/onokumus/Bootstrap-Admin-Template.git yourfoldername
    $ cd yourfoldername
    $ git submodule init
    $ git submodule update
    $ npm install
    $ grunt
    $ cd dist/
    $ open index.html
